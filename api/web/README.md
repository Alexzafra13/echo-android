# Echo Frontend

Interfaz web para Echo Music Server.

## Comandos

```bash
pnpm dev      # Desarrollo
pnpm build    # Build
pnpm preview  # Preview
pnpm test     # Tests
```

## Estructura

```
src/
├── features/   # Módulos (auth, player, library, playlists, admin...)
├── shared/     # Componentes, hooks, services, store
└── app/        # Root, routing
```

## Stack

React, Vite, TypeScript, Tanstack Query, Zustand, Wouter
